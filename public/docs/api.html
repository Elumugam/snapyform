V1 API Documentation
Build powerful integrations with SnapyForm

Authentication

All requests must include your API key in the x-api-key header.

curl -H "x-api-key: YOUR_API_KEY" https://snapyform.com/api/v1/tables

Tables

GET /api/v1/tables

List all tables you have access to.

GET /api/v1/tables/{tableId}

Get table schema and metadata.

GET /api/v1/tables/{tableId}/records

Fetch records from a table. Supports search, sort, and filter query params.

POST /api/v1/tables/{tableId}/records

Create a new record in the table.
